# Figata Order

## Αρχιτεκτονικό Σχέδιο & Mockups

**Μάρτιος 2026**

---

## Τι χτίζουμε

Μια custom εφαρμογή online παραγγελιών φτιαγμένη για το Figata Café. Ο πελάτης μπαίνει από link ή σκανάρει QR code, παραγγέλνει από ένα mobile-friendly μενού, και η παραγγελία εμφανίζεται real-time στο admin panel.

Τρέχει ως **Progressive Web App (PWA)** — ο πελάτης μπορεί να την εγκαταστήσει στο κινητό του σαν κανονική app, χωρίς App Store ή Google Play.

---

## Αρχιτεκτονική Εφαρμογής

```
┌──────────────────┐           ┌──────────────────────┐
│  Customer App    │ ◄───────► │   API (Next.js)      │
│  Mobile PWA      │           └──────────┬───────────┘
└──────────────────┘                      │
                              ┌───────────┴────────────┐
                    ┌─────────▼──────────┐  ┌──────────▼──────────┐
                    │  PostgreSQL DB     │  │    Admin Panel      │
                    │  (Supabase Cloud)  │  │    (web app)        │
                    └────────────────────┘  └─────────────────────┘
```

**Stack:** Next.js · TypeScript · PostgreSQL (Supabase) · Stripe · Vercel

**Hosting:** Vercel + Supabase Cloud — zero server maintenance, 99.9% uptime

## 1. Customer App

### Κατάλογος & Μενού

- Κατηγορίες με tabs (Καφέδες, Ροφήματα, Σνακ, Προϊόντα Σύκου κ.λπ.)
- Αναζήτηση προϊόντων & φίλτρα διατροφής (Vegan, Χορτοφαγικό, Χωρίς Γλουτένη κ.λπ.)
- Πληροφορίες αλλεργιογόνων ανά προϊόν
- Carousel δημοφιλών προϊόντων
- Δίγλωσσο: Ελληνικά & Αγγλικά

### Επιλογές & Παραλλαγές Προϊόντων

- Παραλλαγές ανά προϊόν: Μέγεθος, Γάλα, Ζάχαρη, Extras
- Υποχρεωτικές ή προαιρετικές επιλογές
- Επιπλέον χρέωση ανά επιλογή (π.χ. +0,30€ για φυτικό γάλα)

### Καλάθι & Checkout

- Πλήρης διαχείριση καλαθιού με εμφάνιση modifiers
- Στοιχεία πελάτη + σημειώσεις παραγγελίας
- Πληρωμή με κάρτα (Stripe) ή μετρητά κατά την παραλαβή
- Σελίδα επιβεβαίωσης με αριθμό παραγγελίας

### Παρακολούθηση & Ιστορικό

- Real-time tracking κατάστασης: Νέα → Αποδεκτή → Ετοιμάζεται → Έτοιμη
- Ιστορικό παραγγελιών ανά πελάτη

### QR Code Παραγγελία

- Ο πελάτης σκανάρει το QR, κατεβάζει την εφαρμογή και παραγγέλνει με λίγα απλά βήματα.

## 3. Πληρωμές & Τιμολόγηση

### Stripe

- Online πληρωμές με κάρτα (Visa, Mastercard, Amex)
- Τα χρήματα κατατίθενται απευθείας στον τραπεζικό σου λογαριασμό
- 3D Secure & SCA compliant — PCI DSS compliant (τα στοιχεία κάρτας δεν αγγίζουν ποτέ τον server)

### ΑΑΔΕ myDATA

- Αυτόματη διαβίβαση παραστατικών στην ΑΑΔΕ (XML)
- Δημιουργία, αρχειοθέτηση & ακύρωση παραστατικών
- ΑΑΔΕ Mark & UID, QR code επαλήθευσης

## 4. Roadmap

Αυτές οι δυνατότητες δεν είναι στο αρχικό scope, αλλά η αρχιτεκτονική της εφαρμογής τις υποστηρίζει και μπορούν να προστεθούν εύκολα σε επόμενη φάση.

### Delivery με δικό σου οδηγό

Ο πελάτης θα μπορεί να επιλέγει delivery αντί για takeaway. Ο δικός σου οδηγός λαμβάνει τις παραγγελίες για delivery και ο πελάτης παρακολουθεί την κατάσταση σε real-time.

### Multi-branch υποστήριξη

Αν αποφασίσεις να ανοίξεις δεύτερο κατάστημα, η εφαρμογή επεκτείνεται ώστε κάθε κατάστημα να έχει το δικό του μενού, προσωπικό και ρυθμίσεις — διαχειρίζεσαι τα πάντα από έναν κεντρικό πίνακα.
